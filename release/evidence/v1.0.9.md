# clickhousex v1.0.9 Release Evidence

Date: 2026-06-19

## Scope

This release hardens the production release gate around the existing ClickHouse client surface:

- CI/CD workflows for normal release readiness and factory evidence collection.
- `make release-check` as the L2-T3 production release gate.
- `make test-coverage` as an exact 100.0% coverage gate.
- Machine-readable L2 evidence under `.agent/evidence`.
- Local dev ClickHouse integration and short soak verification from environment-supplied configuration.

## Release Decision

- `release_level_actual`: `L2-T3`
- `release_allowed`: `true`
- `factory_grade`: `false`

`make release-check` is the production release gate for v1.0.9. `make factory-check` remains a stricter L2-T4 gate and must not pass until factory-grade evidence exists.

## Remaining Factory-Grade Gaps

- Production-duration multi-hour live soak evidence is not archived.
- External consumer rollout evidence is not archived.
- Factory-grade release archive has not been produced from a passing factory workflow.
